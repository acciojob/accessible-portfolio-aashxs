Accessible Portfolio

Create an accessible portfolio using ARIA roles to enhance accessibility for users with disabilities.

Instructions:

Bind the following ARIA roles to HTML tags:

Role Tag
imgimg
headingh1
listul
listitemli
navigationnav

Requirements

  1. Include a heading, navigation bar, image, brief description, and a list with at least three items.
  2. All HTML tags must have the respective ARIA roles from the table.
  3. The document must have a proper title and linked CSS file.
  4. The img tag must include an alt attribute.
